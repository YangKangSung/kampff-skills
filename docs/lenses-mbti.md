# MBTI lens (fun / heuristic)

**Not science, not diagnosis.** Color + soft engage-cost only.

## Default policy

| Context | Default |
|---------|---------|
| Community member pipeline | **ON** |
| Workplace mail/meeting batch | OFF unless requested |
| Operator disables lenses | OFF |

## Hard rules

- Title: **「MBTI (fun · low validity)」**
- Prefer leans over hard 4-letter when thin evidence
- Each letter needs ≥1 quote or `confidence: low`
- Never sole reason for `avoid` / hiring / medical claims
- If conflicts with L1, prefer L1 for distance

## Dichotomies (suggestive)

| Axis | Pole A | Pole B |
|------|--------|--------|
| E / I | public, rapid, audience | sparse, long alone-form |
| S / N | concrete steps, tickets | systems, structure, metaphor |
| T / F | criteria, blunt correctness | values, harmony |
| J / P | plans, closed decisions | options open, late pivots |

Keirsey one-liner optional: NT Analyst · NF Diplomat · SJ Sentinel · SP Explorer.

## Output block

```markdown
### MBTI (fun · low validity)
- **Type guess:** …
- **Leans:** E/I · S/N · T/F · J/P
- **Engage cost hint (soft):** …
- **Evidence:** quote · source
- **confidence:** low | medium
```
