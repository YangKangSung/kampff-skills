# CIA / KGB-style lens (public analytic tradecraft)

**Public methods only.** Form inspired by open literature (e.g. Heuer; public SAT primers) and historical **dossier card rhythm** — not classified doctrine, not real service affiliation.

**Not:** covert ops, stalking, blackmail, illegal collection.  
**Yes:** open-text hypotheses for **distance / engage-cost**.

## Default policy

| Context | Default |
|---------|---------|
| Community member pipeline | **ON** |
| Workplace batch | OFF unless requested |
| Operator disables lenses | OFF |

## Hard rules

- ≥3 ACH hypotheses  
- Claim → quote or tagged assumption  
- Distance support only — not legal/medical judgment  
- “KGB/CIA style” = **report structure**, never claimed affiliation  

## Technique order

| Step | Name |
|------|------|
| T1 | Baseline → deviation |
| T2 | Drivers 0–3 (status, control, belonging, autonomy, resource) |
| T3 | Key assumptions check |
| T4 | ACH matrix |
| T5 | Future indicators |
| T6 | Bias checklist |
| T7 | Persona sheet + dossier card |
